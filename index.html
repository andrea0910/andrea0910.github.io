01  /  Research · NLP & Personalization

Evaluating User-Style Adaptation for Professional Text Generation

AI writing tools often sound like everyone and no one at once. What is the minimum information needed to make a model write like you — and which stylistic details actually matter?

Extended HyPerAlign with an 11-field style schema and TF-IDF vocabulary extraction. The optimized three-field schema — greeting style, punctuation habits, distinctive voice — closed over half the gap to human performance using a single writing sample.

02  /  Research · Finance & Algorithms

Stochastic Routing for Cross-Border Payments

International money transfers lose value at every hop — different banks charge different fees, at unpredictable times. This project built a routing model that finds the cheapest path through that uncertainty.

Built a greedy stochastic routing model simulating bank-specific FX spreads, selecting multi-hop routes via Monte Carlo lookahead. Consistently outperforms direct conversion. Core insight: treat uncertainty as a design input, not an obstacle.

Read paper
Skills
Structured ambiguity — modeled a problem with no single right answer and made it testable
Systems thinking — designed for multi-node complexity, not just point-to-point optimization
Tradeoff reasoning — speed vs. cost vs. reliability, quantified and compared
03  /  Senior Capstone · Data & Incentive Design

Data-Driven Approaches for Incentivizing Sales Performance

Insurance sales have been declining. This project asked why — then built a simulation to design incentive structures that actually change behavior, not just measure it.

Built a Monte Carlo simulator for forecasting agent outcomes under different incentive structures — run compensation experiments before deploying them. Quantitative rigor applied to a human question: how do you motivate people doing hard, uncertain work?
